Violations Lib

This is a library for parsing report files from static code analyzis.

It supports:

  • PMD
  • Findbugs
  • Checkstyle
  • CSSLint
  • JSHint

Very easy to use with a nice builder patternn

  List<Violation> actual = violationsApi() //
    .withPattern(".*/findbugs/.*\\.xml$") //
    .inFolder(rootFolder) //
    .findAll(FINDBUGS) //
    .violations();

Developer instructions

To build the code, have a look at .travis.yml.

To do a release you need to do ./gradlew release and release the artifact from staging. More information here.
